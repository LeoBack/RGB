RGB

RGB Controller.

## Project structure ##

**Software:**

- IDE: Visual Studio 2010
- Language: C#
- Requirements.
	1. Windows XP, 7, 8
	2. Framework 4
	3. All that supports Framework 4

The idea is to make the code compatible with Linux, using Mono C # or something similar.

**Hardware:**

Altium Designer Release 10 

**Firmware:**

- IDE: MPLABX 
- Compiler: xc8
- Microcontroller: PIC 16F873

**Simulation:**

Proteus version 7
